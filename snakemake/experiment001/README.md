my-pipeline-project/

│   ├── config.yaml          # Configuration for Snakemake
│   └── samples.csv          # Input metadata (optional)

│   └── results/             # Output files generated by Snakemake

│   ├── main.py              # Main Streamlit app
│   ├── utils.py             # Helper functions (e.g., parsing logs)
│   └── pages/               # Multi-page Streamlit layout (optional)
├── .gitignore
├── README.md
└── requirements.txt         # Python dependencies for the Streamlit app
